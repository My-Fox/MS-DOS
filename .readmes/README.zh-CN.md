# MS-DOS v1.25 和 v2.0 的源代码
这个仓库存放了MS-DOS v1.25 和 MS-DOS v2.0 的源代码和编译好的可执行文件。

我们在 [2014 年 3月 25日于计算机历史博物馆 (Computer History Museum)](http://www.computerhistory.org/atchm/microsoft-ms-dos-early-source-code/) 也分享过一样的档案。现在把这些档案放在这个仓库中，让大家可以更方便地搜索，以及在其他文章或工作中参考，也让对早期 PC 操作系统有兴趣的人可以进行探索和实验。

# 贡献!
这些源代码存放在这个仓库的目的是要留一个历史记录。因此请**不要**发送任何会修改源代码的 Pull Requests 过来。但自行 fork 这个仓库并做自己的实验是没有问题的 😊 。

但如果你想要新增或者修改一些和源代码无关的档案（比如这份 README ) ，请发送 Pull Requests 过来，我们会审核并考虑采用。
